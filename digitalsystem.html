Digital Design System

At Touch of Modern (TOMO), an e-commerce website for modern men, we release promotions and sales via daily emails and ads to all platforms. As the sole designer of these releases, I created the following system to streamline this process for both myself and other teams.

Role: Digital Designer

01. Style Guides

The daily promotions and sales release is a collaboration between various teams. Thus, I created the following style guides to help them determine the appropriate information to give to me (e.g., banner sizes, copy character limits, photo layouts).

Design Guide
Email Style Guide
Promotional Banners Style Guide
Watch Content Banner Guide
Banner Style Guide for Photo Team

02. Templates

After all teams have done their parts, a design request is sent to me to create all the necessary creatives. To make the process more efficient, I have created templates that allow me to produce assets more efficiently.

Promotional and Email Banner Template
Digital Ads Template

03. Feedback

After I create a few drafts, there are multiple iterations of feedback from marketing and merchandising. At this step, we discuss and finalize which design and layout works best.

Promotional Banner Drafts

04. Final Checks & Deliverables

One final round of feedback is done by multiple people to ensure that the copy and banner sizes are correct before exporting. After the files are exported, we make sure that all banners are displayed correctly on all platforms. If there are no errors, then woo hoo, we are done!
